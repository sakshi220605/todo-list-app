# To-Do List Application (CRUD)

## Description
This is a simple to-do list web application
built using HTML,CSS, and Vanilla JavaScript.

## Features
- Add new tasks
- Mark tasks as completed
- Delete tasks
- Prevent empty tasks from being added
- Dynamic DOM manipulation without page reload

## Technologies Used 
- HTML
- CSS
- JavaScript (Vanilla)

## How to Run
1. Download the project
2. Open index.html in a browser

## Output
The application allows users to manage daily
tasks easily.
